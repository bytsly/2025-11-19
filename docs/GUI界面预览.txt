═══════════════════════════════════════════════════════════════════════
                          界面区域说明
═══════════════════════════════════════════════════════════════════════

【系统信息区域】
├─ 系统名称: 显示系统名称
├─ 服务器地址: 显示本地和局域网访问地址
│  • 本地地址: http://localhost:5000
│  • 局域网地址: http://192.168.x.x:5000
└─ 运行状态: 实时显示服务运行状态
   • ● 运行中 (绿色) - 服务正常运行
   • ● 已停止 (红色) - 服务已停止

【控制面板区域】
第一排 - 服务控制:
├─ [▶ 启动服务] 按钮: 启动Web服务器
├─ [■ 停止服务] 按钮: 停止Web服务器
└─ [↻ 重启服务] 按钮: 重启Web服务器

第二排 - 快速访问:
├─ [🏠 打开首页] 按钮: 在浏览器打开系统首页
├─ [⚙ 管理后台] 按钮: 在浏览器打开管理界面
├─ [📊 投票页面] 按钮: 在浏览器打开投票页面
└─ [🎰 抽奖页面] 按钮: 在浏览器打开抽奖页面

【运行日志区域】
├─ 日志文本框: 
│  • 滚动显示所有系统日志
│  • 支持彩色分级显示
│  • 支持鼠标滚轮滚动
│  • 支持Ctrl+F搜索
│  • 支持Ctrl+C复制
│
└─ 日志工具栏:
   ├─ [清空日志] 按钮: 清除所有日志记录
   └─ [导出日志] 按钮: 保存日志到txt文件

═══════════════════════════════════════════════════════════════════════
                          日志颜色说明
═══════════════════════════════════════════════════════════════════════

[SUCCESS] - 绿色文本
  示例: [2024-10-17 10:00:00] [SUCCESS] 服务启动成功！
  用途: 表示操作成功

[INFO] - 黑色文本
  示例: [2024-10-17 10:00:00] [INFO] 客户端已连接
  用途: 普通信息提示

[WARNING] - 橙色文本
  示例: [2024-10-17 10:00:00] [WARNING] 正在停止服务...
  用途: 警告信息

[ERROR] - 红色文本
  示例: [2024-10-17 10:00:00] [ERROR] 端口被占用
  用途: 错误信息，需要处理


═══════════════════════════════════════════════════════════════════════
                          操作流程示意
═══════════════════════════════════════════════════════════════════════

【标准使用流程】

1. 启动程序
   ↓
   [窗口打开，显示初始界面]
   
2. 点击 [▶ 启动服务]
   ↓
   [日志显示启动信息]
   [状态变为 ● 运行中]
   [服务器地址显示]
   [功能按钮全部可用]
   
3. 点击 [⚙ 管理后台]
   ↓
   [浏览器自动打开管理界面]
   [添加候选人、配置系统]
   
4. 点击 [📊 投票页面] 或 [🎰 抽奖页面]
   ↓
   [浏览器打开对应功能]
   [开始使用投票/抽奖功能]
   
5. 使用完毕后点击 [■ 停止服务]
   ↓
   [服务停止]
   [状态变为 ● 已停止]
   
6. 关闭窗口
   ↓
   [如果服务运行中，会提示确认]
   [程序退出]


═══════════════════════════════════════════════════════════════════════
                          快捷操作提示
═══════════════════════════════════════════════════════════════════════

在日志文本框中:
  • Ctrl + A     : 全选所有日志
  • Ctrl + C     : 复制选中的日志
  • Ctrl + F     : 查找文本（部分编辑器支持）
  • 鼠标滚轮     : 上下滚动日志
  • 点击日志框   : 可以选择文本

窗口操作:
  • 拖动标题栏   : 移动窗口位置
  • 拖动边框     : 调整窗口大小
  • 双击标题栏   : 最大化/还原
  • Alt + F4    : 关闭窗口


═══════════════════════════════════════════════════════════════════════
                          注意事项
═══════════════════════════════════════════════════════════════════════

1. 首次启动可能需要3-5秒，请耐心等待

2. 在点击"打开XXX"按钮前，请确保服务已启动（状态为运行中）

3. 日志过多时可能影响性能，建议定期清空

4. 修改.env配置后，需要点击"重启服务"才能生效

5. 关闭窗口前建议先停止服务

6. 如遇到问题，请导出日志进行分析

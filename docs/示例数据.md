# 示例数据说明

## Excel/CSV 导入模板

创建一个Excel文件，包含以下列：

### 基本格式（最少1列）

| 姓名   | 描述         |
|--------|-------------|
| 张三   | 优秀员工     |
| 李四   | 销售冠军     |
| 王五   | 技术专家     |
| 赵六   | 服务之星     |
| 钱七   | 创新达人     |

### 最简格式（仅姓名）

| 姓名   |
|--------|
| 张三   |
| 李四   |
| 王五   |
| 赵六   |
| 钱七   |

## 保存说明

1. 将上述内容复制到Excel中
2. 第一行为列标题，从第二行开始是数据
3. 保存为 `.xlsx`、`.xls` 或 `.csv` 格式
4. 在管理后台选择"文件导入"上传

## OCR识别建议

如果使用照片识别功能：

1. 确保照片清晰，光线充足
2. 姓名文字大小适中（不要太小）
3. 白底黑字效果最佳
4. 姓名为2-4个中文字符
5. 避免手写字（打印体识别率更高）

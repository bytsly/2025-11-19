# 📱 拍照添加候选人 - 快速指南

## 5秒看懂

```
点击"拍照添加" → 拍照/选图 → 输入姓名 → 确认 ✅
```

---

## 操作步骤

### 1️⃣ 打开管理后台
```
http://localhost:5000/admin
登录账号密码
```

### 2️⃣ 进入候选人管理
```
点击"👥 候选人管理"标签
```

### 3️⃣ 点击拍照添加
```
点击"📱 拍照添加"按钮
```

### 4️⃣ 上传照片
**手机端**：
- 点击上传区域
- 自动调起相机
- 拍照

**电脑端**：
- 点击上传区域  
- 选择照片文件
- 确认

### 5️⃣ 输入信息
```
姓名：张三（必填）
描述：优秀员工（可选）
```

### 6️⃣ 确认添加
```
点击"✅ 确认添加"
完成！
```

# 📱 候选人界面二维码 - 快速指南

```
1. 进入候选人管理 → 2. 点击"生成管理后台二维码" → 3. 扫码访问 ✅
```

---

## 详细步骤

### 1️⃣ 打开候选人管理
```
登录管理后台 → 点击"👥 候选人管理"
```

### 2️⃣ 生成二维码
```
找到"📱 扫码访问管理后台"区域
点击"📡 生成管理后台二维码"按钮
```

### 3️⃣ 扫码使用
```
手机扫描二维码
打开管理后台登录页
输入账号密码登录
```

## 注意事项

⚠️ **需要登录** - 扫码后需输入账号密码  
⚠️ **同一网络** - 设备需在同一WiFi或局域网  
⚠️ **IP变化** - IP变化后需重新生成二维码

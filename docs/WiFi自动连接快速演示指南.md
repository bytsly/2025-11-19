## 📞 快速支持

### 紧急问题处理

#### 问题1: 热点无法启动
```
诊断步骤:
1. 检查是否以管理员权限运行
2. 确认主机WiFi适配器正常
3. 查看系统日志错误信息
4. 尝试重启电脑

备用方案:
使用"仅投票二维码"，让用户连接现有WiFi
```

#### 问题2: 大量用户无法访问
```
诊断步骤:
1. 检查服务器是否运行
2. 确认热点IP地址正确
3. 查看连接设备数是否超限
4. 检查网络带宽

备用方案:
增加服务器资源，或分批投票
```

#### 问题3: 投票数据异常
```
诊断步骤:
1. 检查数据库文件
2. 查看最近投票记录
3. 确认是否有重复投票
4. 检查后台日志

处理方法:
数据看板 → 查看投票历史 → 验证数据
```

# 🚀 方案2快速开始指南

## ⏱️ 5分钟部署双二维码投票系统

---

## 📋 准备清单

### 硬件要求
- ✅ 电脑一台（Windows 10+）
- ✅ 投影仪或大屏幕（可选）
- ✅ WiFi网卡（支持热点功能）

### 软件要求
- ✅ Python 3.7+
- ✅ 投票抽奖系统（已安装）
- ✅ 浏览器（Chrome/Edge推荐）

### 网络要求
- ✅ 能创建WiFi热点
- ✅ 支持至少50个设备连接

---

## 🎯 3步快速部署

### 步骤1：启动系统（1分钟）

```bash
# 方式A：使用GUI界面（推荐）
双击运行：启动GUI.bat

# 方式B：命令行启动
cd d:\编程\源码包\投票抽奖系统
python gui_app.py
```

**操作**：
1. 等待GUI窗口打开
2. 点击 `▶ 启动服务` 按钮
3. 看到 "● 运行中" 绿色标志即成功

✅ **成功标志**：窗口显示服务器地址

---

### 步骤2：创建WiFi热点（2分钟）

**方式A：管理后台创建**

```bash
1. 浏览器打开：http://localhost:5000/admin
2. 登录账号：
   用户名：admin
   密码：admin123
3. 点击"网络设置"标签
4. 填写热点信息：
   SSID：投票抽奖系统
   密码：12345678
5. 点击"🚀 创建热点"
6. 等待3-5秒，看到"运行中"状态
```

**方式B：使用批处理（更快）**

```bash
双击运行：启动WiFi热点.bat
```

✅ **成功标志**：状态显示"● 运行中"，已连接设备数为0

---

### 步骤3：生成双二维码（1分钟）

**方式A：首页生成（推荐）**

```bash
1. 浏览器打开：http://localhost:5000/
2. 选择：⦿ 🔗 WiFi + 投票双二维码
3. 确认密码：12345678
4. 点击"生成二维码"
5. 双二维码自动显示
```

**方式B：管理后台生成**

```bash
1. 访问：http://localhost:5000/admin
2. 进入"网络设置"
3. 点击"🔗 WiFi+投票组合"按钮
4. 双二维码并排展示
```

✅ **成功标志**：看到左右两个二维码，带序号①②

## 🛠️ 故障排除

### 问题1：热点创建失败

**症状**：点击创建后显示错误

**解决**：
```bash
1. 确认以管理员权限运行
2. 检查WiFi适配器是否正常
3. 关闭可能冲突的程序
4. 重启电脑后重试
```

---

### 问题2：二维码生成失败

**症状**：点击生成后显示"热点未启动"

**解决**：
```bash
1. 检查热点是否真的在运行
2. 管理后台 → 网络设置 → 检查状态
3. 如果已停止，重新创建热点
4. 确认WiFi名称为"投票抽奖系统"
```

---

### 问题3：用户扫码无反应

**症状**：手机扫描二维码没有提示

**解决**：
```bash
设备检查：
1. 确认手机系统版本（Android 10+ / iOS 11+）
2. 使用系统原生相机（不是第三方应用）
3. 确保光线充足

二维码检查：
1. 二维码尺寸是否足够大
2. 是否有遮挡或模糊
3. 重新生成二维码试试

备用方案：
显示WiFi名称和密码，让用户手动连接
```

---

### 问题4：连接WiFi后打不开页面

**症状**：WiFi连上了，但访问不了投票页面

**解决**：
```bash
1. 检查服务器是否在运行
   - GUI窗口显示"● 运行中"
   
2. 确认热点IP地址
   - 管理后台 → 网络设置 → 查看IP
   - 应该是：192.168.137.1
   
3. 手动访问试试
   - 浏览器输入：http://192.168.137.1:5000/vote
   
4. 重启热点
   - 停止热点 → 等3秒 → 重新创建
```
